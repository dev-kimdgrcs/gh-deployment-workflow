## What I've Learned So Far

- Bash scripting (variables, pipes, awk, cron)
- Linux server administration (SSH, users, permissions)
- Server monitoring (CPU, memory, disk)

## Tools I've Used

I set up a **DigitalOcean** droplet, hardened it with:

1. Key-only SSH authentication
2. Disabled root login
3. Dedicated user with `sudo`

Then I wrote scripts like `server-stats.sh` to monitor it.
